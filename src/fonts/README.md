## Redaction (display) — self-hosted
- Diseñada por Titus Kaphar, Reginald Dwayne Betts, Jeremy Mickel y Forest Young
  para el proyecto/exposición *Redaction* (MoMA PS1) sobre el sistema penal de EE.UU.
- Licencia: **SIL Open Font License 1.1** (OFL).
- Archivos: `Redaction-Regular.woff2`, `Redaction-Bold.woff2`, `Redaction-Italic.woff2`.
- Es la voz display de la marca "Sin Tachar" (concepto: censura/desclasificación).
